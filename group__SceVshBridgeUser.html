◆ vshIoUmount()

int vshIoUmount ( SceVshMountId  id,
int  force,
int  unk2,
int  unk3 
)

Unmount a partition.

Warning
Unmounting system partitions can cause system instability, even if you remount them back. For example, FIOS overlays used by the Shell won't get recreated and would effectively render the Shell unusable
Parameters
[in]id- Mount ID (one of SceVshMountId).
[in]force- 1 to force unmount, 0 otherwise.
[in]unk2- Unknown, set to 0.
[in]unk3- Unknown, set to 0.
Returns
>= 0 on success, < 0 on error.
